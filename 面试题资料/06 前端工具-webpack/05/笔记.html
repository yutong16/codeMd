插件：html-webpack-plugin ***参考网站：https://www.cnblogs.com/wonyun/p/6030090.html 使用步骤： 1. 下载 cnpm install html-webpack-plugin -S 2. 引入 var HtmlWebpackPlugin = require('html-webpack-plugin'); 3. 配置 plugins: [ new HtmlWebpackPlugin() ] 4. 参数说明 4.1 指定模版页面 template:'./src/index.html' 4.2 修改默认输出文件名 filename:'xx.html' 4.3 允许插入到模板中的一些chunk chunks:['jquery','xx'] 4.4 配置不允许注入的thunk excludeChunks:['xx'] 5. 多页面打包 需要创建多个new HtmlWebpackPlugin
